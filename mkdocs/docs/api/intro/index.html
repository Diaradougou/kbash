    api

    The API is defined by - ENTRYPOINT - VAR_PREFIX

    Variable Value

    VAR_PREFIX_KBASH_COMMAND $KBASH/api/util/kbash/state.sh

    api/boot.sh

    boot.sh is invoked with 4 positional arguments

    . <path-to-project-local-activate.sh> \
  "ENTRYPOINT"                      \
  "VAR_PREFIX"                        \
  "USER_UTIL_LOAD_LIST"               \
  "SYSTEM_UTIL_LOAD_LIST"

    About

    • ENTRYPOINT The shell prefix is the command line entry point used to access this environment. It is also used to scope all functions loaded into the shell.
